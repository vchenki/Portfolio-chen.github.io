专注盲盒

一个用反馈回路驱动专注习惯的游戏化工具，将任务抽取、计时专注与随机奖励整合为闭环体验。

游戏机制设计 PWA Chrome Extension 用户激励

Problem & Solution

Problem

传统的番茄工作法依赖自制力维持，但缺乏即时的正向反馈，用户往往在坚持几天后失去动力。市面上的多数专注应用只是简单的计时器，无法形成持续的使用习惯。

Solution

将任务选择本身变成一个有趣的决策行为（盲盒抽取），配合计时结束后的随机奖励，构建「任务投入 → 计时完成 → 即时激励」的完整反馈闭环。
